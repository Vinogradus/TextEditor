Программа TElikeVIM.exe представляет собой текстовый редактор.
Экран разбит на четыре части: основная - для ввода текста, ниже идут окна для ввода комманд
и для вывода информации о текущем режиме работы, самая нижняя часть служит для вывода имени файла,
с которым ведётся работа (работа может вестись и без открытия файла).

Начальный и основной режим - режим навигации. В нём можно перемещатся по напечатанному тексту.
Комманды подаются с английской раскладкой клавиатуры.

Комманды режима навигации:
Стрелки вверх/вниз/влево/вправо - соответственное перемещение курсора
^ - пеермещение курсора в начало строки
$ - пеермещение курсора в конец строки
w - перемещение курсора в конец слова справа от курсора
b - перемещение курсора в начало слова слева от курсора
gg - пееход на начало текста
G - переход в конец текста
NG - (где N - число) переход на строку N
PG_UP - переход на экран вверх
PG_DOWN - переход на экран вниз
x - удалить символ справа от курсора
diw - удалить слово под курсором
dd - вырезать текущую строку (можно будет вставить)
y + Enter - копировать строку
yw - копировать слово под курсором
p - вставить
rA - (вместо A может быть другой символ) заменить символ под курсором

Из режима навигации можно перейти в другие режимы при помощи следующих комманд:
i - обычный переход в режим ввода текста слева от курсора
I - переход в начало строки и активация режима ввода
A - переход в конец строки и активация режима ввода
S - Удаление содержимого строки и переход в режим ввода
: - режим комманд
Shift + F5 - HEX режим
? - переход в режим поиска, поиск осуществляется до начала текста
/ - переход в режим поиска, поиск осуществляется до конца текста

Комманды режима ввода:
backspace - удалить символ слева
delete - удалить символ справа
стрелки, PG_Up, PG_DOWN - аналогично режиму навигации
ESC - выход из режима ввода в режим навигации

Комманды режима комманд:
o filename.txt - открыть filename.txt, расположенный в той же директории
x - записать в текущий файл и закончить работу с файлом
w - записать в текущий файл
w filename.txt - записать текст в filename.txt
q - завершить работу приложения. Если файл был изменён, то выход возможен только через q!
q! -  завершить работу приложения без сохранения
wq! - Записать в текущий файл и завершить работу приложения
number - переход на строку number
h - выводит снизу справку по командам режима комманд
ESC - выход из режима комманд в режим навигации

При переходе в HEX режим основное пространство делится на две области - отображение hex-байт и текста.
Навигация производится только по области отображения hex-байт.
Комманды HEX режима:
NG - (где N - число) переход на N байт
^xN - (где N - последовательность байт, например, 414141) поиск последовательности байт с начала файла
Так же была введена комманда rA, аналогичная по принципу такой же команде в режиме навигации, меняющая байт
Shift + F5 - выход из HEX режима в режим навигации

Режим поиска:
При включении режима поиска вы вводите строку, которую желаете найти, в коммандное окно. После нажатия Enter
осуществится поиск введёной строки в указанном направлении.
n - повторить поиск в том же направлении
N - повторяет поиск, но меняет направление
